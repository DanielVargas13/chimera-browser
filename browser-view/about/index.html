Chimera

what is chimera?

chimera started as a pet project from my dissatisfaction with the general lack of slick looking user interfaces in current browers. it's like a hypothetical offspring of Chrome, Firefox, and Edge... but open source.

extensions

extensions can be installed in "C:\chimera\extensions" on windows or "/usr/share/chimera-extensions" on linux. they'll have full access to your browser so only install extensions from reliable sources, as you would when installing programs.
